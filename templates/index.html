Capability map

Know what stays local and what is optional.

Core document work runs on-device. If you choose a cloud-backed engine for OCR or AI assistance, ConvertFlow treats it as an explicit accelerator rather than the default path.

Fully local core Merge, split, organize, sign, watermark, redact, convert, and many scan-recovery flows run on your machine.
Local AI option Use Ollama when you want local OCR or document assist without sending files to a hosted API.
Optional cloud assist OpenAI and GitHub Models are available only when you configure them and select those engines.
