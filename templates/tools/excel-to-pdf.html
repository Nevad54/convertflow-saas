Tips

  • Conversion uses Microsoft Excel directly — all colors, charts, merged cells, borders, and formatting are preserved exactly.
  • Each sheet in the workbook becomes its own page in the PDF output.
  • Formulas are evaluated to their current values before conversion.
  • Use Fit to 1 page wide to keep all columns on a single row of pages — useful for wide sheets. Fit entire sheet to 1 page squeezes everything onto one page.
  • Page breaks and print areas already set in the workbook are respected when Scale is set to Auto.
