Tips

  • This tool adds PDF/A-1b XMP metadata to mark the document as archival-compliant.
  • For strict ISO 19005 compliance (embedded fonts, color profiles), a more specialised tool like Ghostscript is needed.
  • Most document management systems accept this output as a valid PDF/A document.
  • The original content, structure, and page count are unchanged.
