Tips

  • Works best on digitally-created PDFs where text is embedded. Scanned PDFs without OCR may produce empty or garbled text.
  • The diff report is a standalone HTML file — open it in Chrome, Firefox, or Edge.
  • Differences are shown with 3 lines of context around each change.
  • For scanned PDFs, run them through PDF to Text with Ollama OCR first, then compare the text files manually.
