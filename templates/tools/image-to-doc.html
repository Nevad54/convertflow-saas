Tips

  • auto tries the available recovery engines in order and falls back to local OCR when cloud engines are not configured.
  • tesseract and ollama keep processing on your machine.
  • Set GITHUB_TOKEN or OPENAI_API_KEY as environment variables only if you want optional cloud OCR engines.
  • ollama runs locally via Gemma 4 and needs ollama pull gemma4:e4b.
  • Use legal style to preserve clause numbering and recitals.
  • Use form style for invoices, tables, and checkboxes.
  • Images are sorted by filename before processing, so name them 01, 02, 03 if order matters.
