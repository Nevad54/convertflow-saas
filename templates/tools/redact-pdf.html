Tips

  • Matching is case-sensitive — add both john smith and John Smith if needed.
  • Use exact phrases: +1 555-0100 only matches that exact string.
  • The tool searches and redacts across every page at once.
  • If a phrase is not found, the download is blocked and you'll see an error — so you know the document was checked.
  • Works on digitally-created PDFs. Scanned PDFs without an embedded text layer won't match.
